What is HTML?

HTML (HyperText Markup Language) is the standard language for creating webpages. It uses tags to define the structure of your content.

Common HTML Tags

  • <html>: The root element of the document.
  • <head>: Contains metadata about the webpage.
  • <body>: Contains the visible content of the webpage.
  • <h1> to <h6>: Heading tags for titles.
  • <p>: Paragraph tag for text.

Example Code

<!DOCTYPE html>
<html>
    <head>
        <title>My First Webpage</title>
    </head>
    <body>
        <h1>Hello, World!</h1>
        <p>This is my first webpage.</p>
    </body>
</html>
